Recommended ESO Addons

This is a curated list of addons I found to be the most helpful in ESO so far. You can either choose to install them all, and enable/disable the ones you like in game, or simply install the ones that do not include an * beside it's name.

Installing Minion

Minion makes installing addons for ESO very simple. It isn't required, but it is the recommended way to install addons into the game. Should you wish to manually install addons, you'll have to download all of the addons, extra dependencies, and extra library files needed then extract them into the following folder: C:\Users\YOUR-USERNAME\Documents\Elder Scrolls Online\live\AddOns. The rest of this section will cover how to install and configure Minion.

  1. Head over to the Minion Download Page and download the 32-bit Version Installer. Save the file to your desktop so it will be easy to find.
  2. Navigate to you desktop, right-click on Minion3.0.5-32bit.exe, then left click on "Run as administrator."
  3. Change the installation folder for Minion. Since all of my games install to my G Drive, I placed it there. You should pick either your C or D drives, depending on your computer. Click here for an image
  4. Uncheck "Launch Minion" on the final page and click "Finish". Click here for an image
  5. Follow the instructions here to allow Minion through the Windows firewall: How to Change Windows 10 Firewall Settings to Allow Certain Programs
  6. Right-click on Minion in your Start menu and choose "Run as administrator" then allow the program to update. Relaunch minion by running it as administrator again.
  7. One the main Minion page, uncheck World of Warcraft from the games list and check "Remember settings" at the bottom, then click contine. Click here for an image.
  8. One the following page, select or deselect the hard drives you need. In my case, I have games installed to C and D, so I left those selected. Check "Remember settings" at the bottom and click continue. Click here for an image.
  9. One the last screen that pops up, click on Elder Scrolls Online and navigate the new window to this location: C:\Users\YOUR-USERNAME\Documents\Elder Scrolls Online\live\AddOns. Be sure to select the correct folder under "Documents".
  10. Install everything that you want from the sections below by clicking on "Find More" and searching for the name of the addon or library.

UI Addons

DarkUI

This addon is a reskin of many of the default ESO textures to a smoother and more subdued UI. DarkUI focuses only on changing textures to improve the look and feel of ESO while being compatible with all other addons and having no performance impact.

Last updated 3 mins ago
Votan's Search Box

This add-on enables the built-in search box of the inventory and decorates it, so that it fits the ESO style much better. This addon can be used stand-alone or with Advanced Filters.t.

Last updated 3 mins ago
Advanced Filters

Subfilters to find exactly what you're looking for! Main inventory, personal bank, guild bank, guild store, vendors, and craft bag support!

Last updated 3 mins ago
Lui Extended

LUI Extended is an addon that adds multiple custom components including: Base Game enhancements, Buff & Debuff Tracking, Combat Info, and much more!

Last updated 3 mins ago
Bag Space Indicator

This is a small add-on that makes a rather minor change to the UI to show both the player and bank free space and maximum size, and how much gold the character has.

Last updated 3 mins ago
S'Rendarr*

Adds the ability to track buffs and debuffs on yourself as well as others, both the standard abilities as well as many special buffs like enchants, gear procs, and cooldowns

Last updated 3 mins ago
Ravalox's Quest Tracker*

Provides A more intuitive QUEST TRACKER than the in-game provided tracker interface. The interface can be left on the screen or hidden via settings or /slash commands.

Last updated 3 mins ago
Miat's Stealth Text*

A simple addon that changes the scale and color of your stealth and disguise state messages in the center of your screen.

Last updated 3 mins ago
Urich's Skill Point Finder*

Provides a graphical interface showing all of the skill points you have (and haven't) earned throughout the entire game. You can show or hide the window by binding a hotkey or by entering the slash command.

Last updated 3 mins ago

Map Addons

Map Pins

Adds just about everything you need on a map to the map. Find you skyshards, lore books (but not eidetic memories), treasure maps and more.

Last updated 3 mins ago
Quest Map

Adds pins for completed and uncompleted quests to the world map. (Including quests you haven't started yet!).

Last updated 3 mins ago
Lore Books

Pins on maps of all books of the game. Divided into 3 filters Unknown Lorebooks, Collected Lorebooks, Eidetic Memory.

Last updated 3 mins ago
Harvest Map

HarvestMap displays resource nodes, chests and fishing spots on map, compass and the 3d world.

Last updated 3 mins ago

Misc. Addons

This addon allows you to use the fast travel functionality of the game in a comfortable and highly costumizable way. All fast travel options you have through your guilds, friends and group are listed in a clear way.
